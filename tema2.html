GRUNDLÆGGENDE WEB

Introduktion til HTML og vs-code

I dette tema blev vi introduceret til HTML ved brug af programmet VS-code. Vi gennemgik HTML i plenum, for at sikre vi alle var med og forstod, hvad der foregik på skærmen. Efterfølgende fik vi adgang til Free Code Camp, hvor vi kunne øve os yderligere og selvstændigt.

Lære at bruge vs-code

Vi fik en opgave, hvor vi skulle kunne vise, hvordan vi havde forstået at bruge HTML. Om vi kunne sætte billede og tekst ind med nogle krav ift. Billedstørrelse og overskrifter. Det handlede om vores kæledyr og vi skulle også lave det som et visitkort. Det var udelukkende HTML, for efterfølgende blev vi introduceret til CSS. Det gjorde vi nemlig ved at tage fat i den første opgave vi lavede i plenum, men nu skulle vi designe den, så det blev pænere. I samme omfang blev vi også præsenteret for designprincipper fra vores bog. For at kunne arbejde med CSS, lærte vi om grid, flex og mediaquery (når vi skulle kode hjemmesiden til mobil også).

Studiestartsprøven

Da vi skulle lave studiestartsprøven, havde vi lært om designprincipper. Primært handlede det om gestaltlove, da vi skulle til at arbejde med CSS for første gang. Generelt skulle vi have fokus på loven og nærhed, da titler og brødtekst og billeder var meget tæt på hinanden på hjemmesiden. Ved at bruge gestaltlovene sammen med CSS, kunne vi vise flere kompetencer. At vi begyndte at lære sproget i CSS at kende, og at vi forstod gestaltlovene og kunne anvende dem i praksis. Vi fik tildelt layoutdiagrammer og wireframes, som vi skulle bruge til at lave layout på hjemmesiden.

Aflevering

For at aflevere denne opgave skulle vi bruge wiseflow for første gang. Hjemmesiden skulle uploades til vores eget domæne, ved brug af filezilla og den mappestruktur vi var blevet introduceret til. Når vi havde afleveret prøven i wiseflow, fik vi senere at vide om vi var var bestået eller ikke bestået.

Opsumering

Generelt var dette tema en stor mundfuld og udfordrende, da vi skulle anvende mange nye redskaber og programmer på kort tid. Men selvom vi også skulle lære om designprincipper samtidig, gav det rigtig god mening i temaet. Det hele gik hånd og hånd, og der var masser hjælp at få både fra lærere og tutorere. Temaet forberedte os godt på næste tema, og selvom det var svært, så var det godt konstrueret og derfor til at komme fint igennem.
